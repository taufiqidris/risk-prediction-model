# ICU Mortality Risk Prediction Model

This project builds an explainable machine learning model to predict ICU mortality risk using tabular clinical data.

The objective is to demonstrate how data science can support clinical decision-making through early risk identification and interpretable machine learning models.

Key focus areas:
- Imbalanced classification
- Model interpretability (SHAP)
- Clinical feature importance
- Risk prediction using tabular healthcare data

---

# Dataset

This project uses the **ICU Death Classification dataset** from Kaggle:

https://www.kaggle.com/datasets/fdemoribajolin/death-classification-icu

The dataset contains anonymised ICU patient features used to predict mortality outcomes.
